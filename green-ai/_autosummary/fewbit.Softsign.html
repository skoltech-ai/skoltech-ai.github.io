fewbit.Softsign¶

class fewbit.Softsign(*, bits: Optional[int] = None)¶

Applies the element-wise function:

\[\text{SoftSign}(x) = \frac{x}{ 1 + |x|}\]
Parameters:

bits – number of bits in gradient approximation: Default: 3

Shape:

  • Input: \((*)\), where \(*\) means any number of dimensions.

  • Output: \((*)\), same shape as the input.

Examples:

>>> m = fewbit.Softsign()
>>> input = torch.randn(2)
>>> output = m(input)

See also

torch.nn.Softsign – Original PyTorch implementation.
