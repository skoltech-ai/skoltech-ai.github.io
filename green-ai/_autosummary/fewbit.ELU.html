fewbit.ELU¶

class fewbit.ELU(alpha: float = 1.0, *, bits: Optional[int] = None)¶

Applies the Exponential Linear Unit (ELU) function, element-wise, as described in the paper: Fast and Accurate Deep Network Learning by Exponential Linear Units (ELUs).

ELU is defined as:

\[\begin{split}\text{ELU}(x) = \begin{cases} x, & \text{ if } x > 0\\ \alpha * (\exp(x) - 1), & \text{ if } x \leq 0 \end{cases}\end{split}\]
Parameters:

  • alpha – the \(\alpha\) value for the ELU formulation. Default: 1.0

  • bits – number of bits in gradient approximation: Default: 3

Shape:

  • Input: \((*)\), where \(*\) means any number of dimensions.

  • Output: \((*)\), same shape as the input.

Examples:

>>> m = fewbit.ELU()
>>> input = torch.randn(2)
>>> output = m(input)
