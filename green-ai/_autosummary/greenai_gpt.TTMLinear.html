greenai_gpt.TTMLinear¶

class greenai_gpt.TTMLinear(d_in: int, d_out: int, rank: int)¶

Creates TTM layer corresponding to Linear layer with weihgts matrix M x N.

Pads initial layer shape up to best approximated shape and create a TTM object.

Parameters:

  • d_in (int) – The first dimension (M) of the weights in the initial Linear layer.

  • d_out (int) – The second dimension (N) of the weights in the initial Linear layer.

Variables:

  • d_in (int) – The first dimension (M) of the weights in the initial Linear layer.

  • d_out (int) – The second dimension (N) of the weights in the initial Linear layer.

  • d_in_ttm (int) – Nearest number to d_in that can be factored by a value not greater than treshhold.

  • d_out_ttm (int) – Nearest number to d_out that can be factored by a value not greater than treshhold.

  • ttm – Object of TTM class.
